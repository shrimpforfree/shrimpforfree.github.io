BeeBot Event Recommendation System

Personalizes nearby events using embeddings with geo and freshness signals.

Role: Backend/iOS Intern (Contributor)

Overview

Built a hybrid event ranking pipeline combining semantic similarity (embeddings) with geo proximity (radius prefilter + distance weighting) and recency decay. Added duplicate suppression and ingest quality gates. Result: faster queries and higher top-K precision on real user traffic.
