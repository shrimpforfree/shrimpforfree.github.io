Overview

I built a hybrid event ranking pipeline combining semantic similarity (embeddings) with geo proximity (radius prefilter + distance weighting) and recency decay. Added duplicate suppression and ingest quality gates. Result: faster queries and higher top-K precision on real user traffic.
